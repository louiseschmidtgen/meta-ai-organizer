# meta-ai-organizer

A meta repository for tracking the work carried out by AI agents. All artefacts — scripts, reports, and prompts — are version-controlled here so every action taken by an agent is auditable and reproducible.

## Repository Structure

```
meta-ai-organizer/
├── scripts/    # Automation and utility scripts run by AI agents
├── reports/    # Output reports documenting agent runs and findings
└── prompts/    # Versioned prompts used to instruct AI agents
```

### `scripts/`
Contains automation scripts, task runners, and utility scripts executed by or on behalf of AI agents. See [`scripts/README.md`](scripts/README.md) for conventions.

### `reports/`
Stores Markdown reports generated by AI agents, capturing objectives, outcomes, and next steps for each run. See [`reports/README.md`](reports/README.md) for the naming convention and template.

### `prompts/`
Holds reusable, version-controlled prompts used to instruct AI agents. Keeping prompts here makes iteration and collaboration straightforward. See [`prompts/README.md`](prompts/README.md) for conventions.

## Getting Started

1. **Add a script** – drop it in `scripts/` (or a relevant subdirectory) and add a brief header comment.
2. **Log a report** – after an agent run, save a report to `reports/` following the `YYYY-MM-DD_<agent>_report.md` naming scheme.
3. **Version a prompt** – store the prompt in `prompts/` so changes are tracked and earlier versions remain accessible via Git history.

# Scripts

This directory contains scripts executed by or on behalf of AI agents.

## Structure

Store automation scripts, task runners, and utility scripts here. Each script should be named descriptively so its purpose is immediately clear.

## Conventions

- Use subdirectories to group related scripts (e.g. `scripts/data-processing/`, `scripts/deployment/`).
- Include a brief comment at the top of each script describing what it does.
- Prefix script filenames with the agent or workflow name where applicable (e.g. `agent1_fetch_data.sh`).

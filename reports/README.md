# Reports

This directory stores reports generated by AI agents documenting their work, findings, and outcomes.

## Structure

Each report should be saved as a Markdown or plain-text file with a clear, dated filename so results are easy to browse chronologically.

## Naming Convention

```
YYYY-MM-DD_<agent-or-task-name>_report.md
```

**Example:** `2024-06-01_data-analysis-agent_report.md`

## What to Include in a Report

- **Date / Timestamp** – when the task was run.
- **Agent / Workflow** – which agent or script produced this report.
- **Objective** – what the agent was asked to do.
- **Summary** – a short paragraph summarising the outcome.
- **Details** – full output, metrics, or findings.
- **Next Steps** – recommended follow-up actions, if any.
